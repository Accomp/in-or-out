# In or Out?
This application is intended with the sole purpose of creating a simple means of asking your friends and family if they are interested in or attending events.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
